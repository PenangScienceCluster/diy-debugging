  4. Remove the battery from the laptop, then start charging the laptop. Do you see LED indicators lighting up?

  9. Plug in a Windows 10 installer and attempt to install the operating system. Does it show an installable hard drive as its target?

    1. No

      • Go to 10

    2. Yes

      • Proceed to installing Windows 10 on the hard drive. It is likely the hard drive is either blank, or the files on it are corrupted.

  14. Do you have access to the password to unlock the encrypted hard drive?

    2. Yes

  19. Open the Windows Update service from the Windows settings app. Can you run updates?

    1. No

      • Check that you are properly connected to the Internet then retry.
